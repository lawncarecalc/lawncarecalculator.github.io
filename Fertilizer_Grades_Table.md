# Help Me Choose a Fertilizer at the Garden Center — Grades by Phosphorus Need

This is the brand-free reference table built into the calculator's "Help me choose a fertilizer at the garden center" section. It is now organized by **phosphorus need** so a resident reads their soil test's P rating (Low / Medium / High) and goes straight to the matching group. Match the N-P-K and WIN on any bag to the closest row — an exact match isn't needed, just something close. Generic profiles reflect what is commonly stocked at Virginia garden centers; **no brands are endorsed.**

| **46-0-0** (urea) | N correction only, P & K adequate | 0% | P1 |
| **32-0-6** (30% coated urea) | Maintenance, steady slow feed | ~30% | P2 |

## How to read WIN correctly

⚠️ **Program (the per-application N limit) is set by WIN as a percent of total nitrogen, not percent of bag weight.** Divide the label's WIN figure by the total N%.

- Example: a bag listing 24% total N and "0.6% water insoluble nitrogen" → 0.6 ÷ 24 = **2.5% of N** → Program 1.
- Example: a 39-0-0 listing "28.5% water insoluble nitrogen" → 28.5 ÷ 39 = **73% of N** → Program 3.

Many low-P and starter grades come in **both** quick-release and slow-release versions, so the "by WIN" entries mean: read the bag's guaranteed analysis to determine the program.

**Slow-release chemistries to look for in the ingredient list:** methylene urea, MESA, ureaform, sulfur- or polymer-coated urea, IBDU, or protein-based organics (feather/blood/meat meal).

## Program thresholds (VCE 430-011 Tables 2–4; Soil Test Notes 17 & 18)

- **WIN < 15% of N → Program 1** (quick-release): max 0.7 lb N/1,000 sq ft per application (cool-season).
- **WIN 15–49% of N → Program 2** (slowly available): max 0.9 lb (cool) / 1.0 lb (warm) per application.
- **WIN ≥ 50% of N → Program 3** (majority slowly available): highest per-application rate, fewest applications.

## Phosphorus cautions

- **Apply P only when your soil test calls for it.** Starter/high-P grades are for new lawns or Low-P soils — on High-P soil they waste money and run off into waterways.
- **Starter and "complete" grades are intentionally high-P** (24-25-4, 18-24-12, etc.) and belong in the new-lawn / Low-P path only, never standard maintenance on adequate soil.
- **Some organic N sources carry phosphorus** (e.g. biosolid-based products ~4% P₂O₅) — disqualified when P tests High.
- **Formulations change yearly.** Treat these as starting profiles; the bag's guaranteed-analysis label is always the authority.

*Sources: VCE 430-011; VCE Soil Test Note 17 (452-717) and Note 18 (452-718). No brands endorsed.*
